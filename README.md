
# react-dial-phone

- Experiment to produce phone sounds using [AudioContext](https://developer.mozilla.org/en-US/docs/Web/API/AudioContext)
- Every key will produce a sound similar to a phone, including the ring tone

Install: 
```
yarn install
```

Run: 
```
yarn start
```
